## ✎📓🗒ˎˊ˗ 𝑪𝒂𝒍𝒄𝒖𝒍𝒂𝒅𝒐𝒓𝒂 𝑨𝒏𝒅𝒓𝒐𝒊𝒅 ⋆.ೃ࿔*:･

Projeto de uma **calculadora simples para Android**, desenvolvida em **Kotlin** utilizando **Android Studio**.  
O aplicativo realiza operações matemáticas básicas com lógica implementada manualmente, sem uso de bibliotecas externas para cálculo.

## ㅤ♡𝗙𝘂𝗻𝗰𝗶𝗼𝗻𝗮𝗹𝗶𝗱𝗮𝗱𝗲𝘀

- ✚ Soma
- — Subtração
- ✘ Multiplicação
- ÷ Divisão
- ✿ Inserção de números (0–9)
- ☆ Ponto decimal
- ☘︎ Botão CE para limpar tudo
- 🗑 Botão para apagar o último número digitado
- ≈ Botão igual para calcular o resultado

## 𝄞⨾𓍢ִ໋ 𝗧𝗲𝗰𝗻𝗼𝗹𝗼𝗴𝗶𝗮𝘀 𝘂𝘁𝗶𝗹𝗶𝘇𝗮𝗱𝗮𝘀

- Kotlin
- Android Studio
- XML (Layouts Android)
- View Binding

## ♡ 𝗘𝘀𝘁𝗿𝘂𝘁𝘂𝗿𝗮 𝗱𝗼 𝗽𝗿𝗼𝗷𝗲𝘁𝗼

**MainActivity.kt**  
Responsável pela lógica da calculadora:
- controle dos números digitados
- operações matemáticas
- atualização do display

**activity_main.xml**  
Arquivo de layout que define:
- display da calculadora
- botões numéricos
- botões de operação

## ⋆˚࿔ 𝗜𝗻𝘁𝗲𝗿𝗳𝗮𝗰𝗲

A interface utiliza um **tema em tons de rosa**, com botões estilizados e bordas personalizadas para o display.

## ✌︎㋡ 𝗙𝘂𝗻𝗰𝗶𝗼𝗻𝗮𝗺𝗲𝗻𝘁𝗼

A calculadora funciona armazenando os números digitados como **String** e convertendo para **Double** no momento do cálculo.  
O aplicativo também controla estados internos para saber se o usuário está digitando o primeiro ou o segundo número.


## ୨ৎ 𝗖𝗼𝗺𝗼 𝗲𝘅𝗲𝗰𝘂𝘁𝗮𝗿 𝗼 𝗽𝗿𝗼𝗷𝗲𝘁𝗼

1. Baixe ou clone o repositório
2. Abra no **Android Studio**
3. Aguarde o Gradle sincronizar
4. Execute em um **emulador Android** ou **celular**
